# game

:video_game: Canvas 2D Game Engine

## Features

- ✅ Unlimited players
- ✅ Independent keyboard controls for each player
- ✅ Support display bounding box
- ✅ Scenes (create, switch between)
- ✅ UI Events on any entity (like player)
